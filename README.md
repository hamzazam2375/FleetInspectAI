FleetInspect AI — Frontend Prototype

This is a React + Vite frontend-only prototype for vehicle return inspections.

Quick start:
1. npm install
2. npm run dev

Features:
- Dashboard, Vehicle list, Return inspection flow
- Image upload with simulated AI findings
- Printable / downloadable inspection report (frontend-only)

Notes: Tailwind CSS is configured; run `npm install` to fetch deps.
